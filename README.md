## Introduction

주요 암호화폐들의 정보들만 쏙쏙 제공하는 서비스입니다.

## Main Feature

- 주요 암호화폐 정보 제공
- 가격 차트 제공
- 가격 변동률 제공
- 다크모드 토글 기능

## Environment

- Language: JavaScript, TypeScript
- Framework: React.js
- CSS: Styled Components
- Server Data State Management: React Query
- Status Management: Recoil
- Other Library: Apex Charts
- Deploy: Github
